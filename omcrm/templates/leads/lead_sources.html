API Usage Information

Lead Import API Endpoint
Required Parameters
  • api_key - The API key shown in the table above
  • affiliate_id - The Affiliate ID from the table above
  • first_name - Lead's first name
  • last_name - Lead's last name
  • email - Lead's email address
Optional Parameters
  • phone - Lead's phone number
  • country - Lead's country
  • company_name - Lead's company name
  • notes - Additional notes
  • funnel_name - Funnel name for tracking
Example API Call (GET)
{{ request.host_url }}api/import_lead?api_key=YOUR_API_KEY&affiliate_id=YOUR_AFFILIATE_ID&first_name=John&last_name=Doe&email=john@example.com
Example API Call (POST JSON)
fetch('{{ request.host_url }}api/import_lead', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json'
    },
    body: JSON.stringify({
        api_key: 'YOUR_API_KEY',
        affiliate_id: 'YOUR_AFFILIATE_ID',
        first_name: 'John',
        last_name: 'Doe',
        email: 'john@example.com'
    })
})
.then(response => response.json())
.then(data => console.log(data));
